1. Generic'ами в Java называют классы, интерфейсы или методы, содержащие в себе типы-параметры. Это позволяет писать программу, не зависящую от конкретных типов данных.
2. Лямбда-выражения (позволяют передавать функции в качестве параметров других функций);
   Функциональные интерфейсы, используемые для работы с лямбда-выражениями;
   Потоки (Streams).
3. Hash-code - целочисленное значение фиксированной длины, которое вычисляется для любого объекта в Java на основе его данных. 
4. Основное отличие стека и очереди заключается в порядке добавления и удаления элементов. Очередь работает по принципу FIFO (first in, first out), когда первый добавленный элемент будет удалятся первым, а стек по принципу LIFO (первым удаляется последний добавленный элемент).
5. Assert - это оператор, используемый для проверки параметров в коде и состояния программы или ее данных. Работает путем вставки утверждения, которое должно оказаться истинным во время выполнения программы.
6. Comparator в Java - это класс, реализующий интерфейс Comparable, используемый для сравнения двух объектов между собой. Comparable содержит метод compare(), принимающий два объекта и возвращающий либо отрицательное число, либо ноль, либо положительное число (этот метод часто используется при сортировке).
